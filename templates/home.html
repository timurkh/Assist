About

There have been several reasons to create this portal:

  • Because I can To get experience of developing web app using modern frameworks and technologies: Google Application Engine (Cloud Firestore, Firebase Auth) + Go + modern vanilla JSVue + Bootstrap.
  • I've got vague feeling that modern web is missing a platform, that would allow to define (or better identify itself) entities which surround us and actually form our life (people, places, events, needs), link them with each other and define actions on top of these entities intersection. Here I try to build working model for my ideas.
  • Being able to create code should definitely have positive day-to-day outcome. Here I will implement automation for certain scenarios that drive me mad for years.
